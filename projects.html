TraduCode

TraduCode is an innovative Chrome extension designed to bridge the gap between Spanish and English programming languages. Developed using modern web technologies like HTML, CSS, and JavaScript, it leverages Chrome's Extension API to seamlessly integrate with the browser. The extension utilizes a comprehensive JSON-based translation dictionary to convert Spanish programming keywords and constructs into their English equivalents in real-time.
